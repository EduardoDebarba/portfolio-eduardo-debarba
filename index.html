Clone do site da Spacex

Data de Realização: Janeiro de 2025

Este projeto consiste em um clone do site oficial da SpaceX, desenvolvido com HTML, SASS, Bootstrap e JavaScript. Recriei todas as páginas do site original, incluindo informações sobre os foguetes Falcon 9, Falcon Heavy, Starship e a cápsula Dragon. Adicionei também uma seção sobre a missão Artemis e outra sobre os planetas do sistema solar.

Utilizei o Bootstrap para garantir a responsividade do site e o SASS para otimizar o desenvolvimento do estilo visual. O JavaScript foi implementado para adicionar interatividade e funcionalidades dinâmicas, proporcionando uma experiência de usuário imersiva e fluída.

Game Minds Quiz

Data de Realização: Março de 2024

O Game Minds Quiz é um jogo de perguntas e respostas sobre o universo dos videogames, desenvolvido com HTML, CSS e JavaScript. O jogo apresenta questões com quatro opções de resposta, sendo apenas uma correta. A cada acerto, o jogador ganha 1 ponto, enquanto erros são contabilizados separadamente, permitindo acompanhar o desempenho ao longo do quiz.

Este projeto explora a interatividade com o usuário por meio de lógica em JavaScript, oferecendo uma experiência dinâmica e envolvente. Além disso, o design foi pensado para garantir uma interface intuitiva e agradável, tornando o quiz acessível para todos os públicos.
